# Mission Dashboard Specification

## Overview
The Mission Dashboard is the primary interface for PathForge, an AI-powered execution intelligence system. Its singular purpose is to help users avoid missing deadlines.

## Columns

### Left Column: Mission Control
- **Mission:** The core goal.
- **Goal Input:** Input field to start a new mission.
- **Deadline:** Deadline selection.
- **Constraints:** Any user constraints.
- **Hours Available:** Time available to commit.
- **Primary Action:** 🚨 Save My Deadline.

### Center Column: Execution Overview
- **Mission Summary:** High-level summary.
- **Execution Status:** Pipeline status.
- **Today's Mission:** Immediate next steps.
- **Upcoming Milestones:** Key milestones.
- **Future Placeholder:** Responsive container for Interactive DAG, Timeline, and Streaming Activity.

### Right Column: AI Intelligence
- **Mission Health:** Overall health status.
- **Success Probability:** Probabilistic chance of hitting the deadline.
- **Biggest Risk:** Top risk identified by the system.
- **Recommended Next Action:** AI suggested next step.
- **AI Explanation:** Explanation of the analysis.

## Core Principles
1. What is my mission?
2. Am I likely to finish?
3. What is putting my deadline at risk?
4. What should I do next?
5. How can I save my deadline?
