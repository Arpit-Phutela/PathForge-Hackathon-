# Responsive Rules

## Desktop (> 1024px)
- **Layout:** Three-column grid layout.
- **Left Column:** Fixed width or relative (e.g., 25%).
- **Center Column:** Expanding main column (e.g., 50%).
- **Right Column:** Fixed width or relative (e.g., 25%).

## Tablet (768px - 1024px)
- **Layout:** Two columns or stacked sections depending on container width.
- **Priority:** Left and Center merge, Right drops below or forms a 2-column grid.

## Mobile (< 768px)
- **Layout:** Single column, stacked vertically.
- **Prioritization:** 
  1. Mission Header
  2. Mission Health / Success Probability
  3. Today's Mission
  4. Biggest Risk
  5. Save My Deadline CTA
- **Hidden / Progressive Disclosure:** Deep analytics hidden behind expandable sections or tabs.
